# memorycard
Memory Card — це проста графічна програма-вікторина, створена з використанням Python та PyQt5.
Користувачу по черзі показуються питання з чотирма варіантами відповідей, з яких лише один є правильним.

Програма підходить для навчання основ:

роботи з PyQt5

об’єктно-орієнтованого програмування

логіки перевірки відповідей

випадкового вибору даних
